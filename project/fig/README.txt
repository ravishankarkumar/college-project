These softwares [fig_v(x)] are produced in versions  with the specifications as follows:

Version1:
This version is developed to find the type of Triangle using length of edges of given figure.

Version2:
This version is developed to find the type of Triangle or quadrilateral using length of edges of given figure.

Version3:

This version is developed to find the type of Triangle using length of edges or coordinates of given figure.
it also find the type of quadrilateral using  length of sides.


Version4:

This version is developed to find the type of Triangle using length of edges or coordinates of given figure.
it also find the type of quadrilateral using given coordinates or given length of sides.
